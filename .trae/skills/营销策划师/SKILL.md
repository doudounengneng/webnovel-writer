## 书名优化

### 高点击书名公式

```
爆款书名 = 核心卖点 + 情绪钩子 + 记忆点

常见公式：

【公式1：身份+冲突】
  "穿越洪荒：我拜入阐教被偏见，三年后震惊三界"
  "重生都市：我靠炒股成为首富"

【公式2：金手指+爽点】
  "系统流：开局签到荒古圣体"
  "悟性逆天：我一眼看穿功法破绽"

【公式3：悬念+反转】
  "我真是大反派？读者都求我别洗白"
  "被宗门抛弃后，我成了魔教教主"

【公式4：对比+碾压】
  "别人修仙我修车，怎么就成了最强器师？"
  "全宗门都觉得我是废材，直到我拿出上古神器"
```

### 书名测试矩阵

```
【书名方案评估】

| 方案 | 核心卖点 | 情绪钩子 | 记忆点 | 预估点击率 |
|------|---------|---------|--------|-----------|
| 方案A | 穿越洪荒 | 被偏见 | 震惊三界 | ★★★★☆ |
| 方案B | 系统流 | 开局签到 | 荒古圣体 | ★★★★★ |
| 方案C | 废材逆袭 | 被抛弃 | 魔教教主 | ★★★☆☆ |

推荐方案：方案B（系统流+开局签到+荒古圣体）
理由：系统流当前热门，开局签到是爽点，荒古圣体有记忆点
```

### 平台适配

```
起点风格：大气/有深度
  "洪荒：从阐教弃子到三界共主"

番茄风格：直白/爽点前置
  "穿越洪荒：开局签到荒古圣体，三界震惊"

飞卢风格：创新/反差
  "洪荒：我靠科学修仙，天道都懵了"

七猫风格：下沉/情绪化
  "被阐教抛弃后，我转身成了洪荒最强"
```

## 简介打磨

### 简介结构

```
【黄金简介公式】

第一段：场景+身份（1-2句）
  例："陈凡穿越洪荒，拜入阐教门下。"
  目标：让读者知道背景

第二段：冲突+困境（1-2句）
  例："因出身低微，被师兄们百般刁难，连师父都对他爱答不理。"
  目标：制造共情

第三段：转折+金手指（1-2句）
  例："直到那天，他觉醒了荒古圣体——"
  目标：给希望

第四段：爽点+期待（1-2句）
  例："从此，废材逆袭，打脸不断。三界震动，万族臣服！"
  目标：刺激点击

第五段：钩子（1句）
  例："ps：没有憋屈，只有爽！"
  目标：消除顾虑
```

### 简介类型

```
【爽文型】
"陈凡穿越洪荒，拜入阐教。因资质平庸，被师兄们当众羞辱。
他忍了。
直到那天，他觉醒了荒古圣体——
一拳，打碎师兄的骄傲。
一掌，拍灭长老的质疑。
一剑，斩断天道的枷锁。
从此，三界震动，万族臣服。
ps：没有憋屈，只有爽！"

【情感型】
"她穿书了，成了恶毒女配。
按照剧情，她会在第三章被男主杀死。
她决定：远离男主，苟到大结局。

'你为什么不按剧本走？'
'因为——我想活着。'
男主愣住了。
后来，他为了她，毁了整个剧本。"

【悬念型】
"我穿成了修仙文里的路人甲。

但我绑定了一个系统：
'恭喜宿主，获得'剧情修改器'。'
我笑了。
'那我把男主改成女的，不过分吧？'
后来，男主（女）把我按在墙上：
'你对我做了什么？'
我：？？？"
```

## 封面策略

### 封面设计原则

```
色彩选择：
  玄幻/仙侠：金色/红色/紫色（大气）
  都市/现代：蓝色/白色/黑色（简洁）
  言情/甜宠：粉色/白色/暖色（温柔）
  悬疑/惊悚：黑色/红色/暗色（紧张）

字体选择：
  标题：粗体/有力量感
  副标题：细体/清晰
  作者名：小字/低调

元素选择：
  主角形象：帅气/美丽/有辨识度
  场景元素：与世界观相关
  特效：适度，不过度
```

### 封面测试

```
【封面A/B测试】

封面A：主角正面+金色特效
封面B：场景全景+红色标题

测试结果：
  封面A点击率：3.2%
  封面B点击率：2.8%

结论：封面A胜出，主角正面形象更有吸引力
```

## 首发策略

### 新书预热

```
【预热方案】

预热期（前7天）：
  1. 确定书名/简介/封面
  2. 存稿至少20章
  3. 在读者群预告
  4. 发布预告帖

首发期（前7天）：
  1. 日更3章（冲量）
  2. 每章都有爽点
  3. 章末强钩子
  4. 积极回复评论

稳定期（第2-4周）：
  1. 日更2章
  2. 开始加更规则
  3. 建立读者群
  4. 收集反馈
```

### 首发平台选择

```
【首发决策树】

作品类型是什么？
  ├── 玄幻/仙侠 → 起点/番茄
  ├── 都市/赘婿 → 番茄/七猫
  ├── 同人/系统 → 飞卢
  ├── 言情/甜宠 → 晋江/番茄
  └── 悬疑/科幻 → 起点/掌阅

目标读者群体？
  ├── 男性为主 → 起点/飞卢
  ├── 女性为主 → 晋江/番茄
  └── 全年龄 → 番茄/七猫

更新能力？
  ├── 日更6000+ → 起点（追读要求高）
  ├── 日更4000+ → 番茄（完读要求高）
  └── 日更2000+ → 七猫（要求较低）
```
