# 网文对比差异分析

将大神的网文与自己网文进行全面对比，识别差异点，学习大神的优点，修正自己的不足。

## 核心功能

1. **全面对比分析** - 世界观构建、人物塑造、情节设计、情感表达、文笔风格五维度对比
2. **差异识别** - 精准定位自己与大神的差距
3. **自动修改** - 直接输出优化后的版本

## 对比分析框架

### 五维对比体系

| 维度 | 分析内容 | 大神标准 | 自检问题 |
|------|---------|---------|---------|
| **世界观构建** | 体系完整性、逻辑自洽、创新性 | 修炼体系完整、世界观有层次感 | 体系是否自洽？是否有漏洞？ |
| **人物塑造** | 主角/配角/反派立体化 | 人设有内在矛盾、行为有逻辑 | 是否工具人？是否有独立动机？ |
| **情节设计** | 爽点分布、节奏把控、钩子设置 | 爽点密集、节奏紧凑、章末有钩 | 爽点是否到位？节奏是否拖沓？ |
| **情感表达** | 情感线推进、情绪拉扯、暧昧/热血 | 拉扯细腻、情绪起伏、克制但到位 | 是否太直白？是否有起伏？ |
| **文笔风格** | 叙事大气、对话真实、描写精准 | 对话像真人、描写有画面感 | 是否废话？对话是否像真人？ |

### Step 3: 差异汇总与优先级

#### 🚦 严重度判定标准

在填入"核心差距/次要差距"前，用以下规则确定每项差距的严重程度：

| 严重度 | 判定条件 | 典型场景 |
|--------|---------|---------|
| **P0 必须改** | 该差距导致读者流失或追读率持续下降 | 世界观自相矛盾、主角OOC、爽点为0的章节 |
| **P0 必须改** | 该差距在后续章节会放大（不改→越写越崩） | 战力体系崩坏、伏笔死线、主线偏离 |
| **P1 建议改** | 影响阅读体验但不致命 | 对话太平、配角扁平、节奏偏慢 |
| **P1 建议改** | 改后会有明显提升但改动成本可控 | 章末钩子太弱、描写缺乏画面感 |
| **P2 可改可不改** | 风格差异而非质量问题 | 你的叙事节奏偏慢但读者不抱怨 |
| **P2 可改可不改** | 改动成本 > 收益 | 重构已完成的整卷结构 |
| **保持（不改）** | 差距虽存在，但正好是你的特色 | 你有独特的幽默感，大神没有 |

⚠️ 判定铁律：
- **函数性差距**（直接导致读者流失/追读率下降）→ P0，二话不说
- **体验性差距**（影响阅读感受但不致命）→ P1，量力而行
- **风格性差异**（没有对错，只是不同）→ 保持，这就是你

**判定流程：**
```
发现差距 → 问：这个差距是否直接导致读者流失？
  ├── 是 → P0
  └── 否 → 问：改动这个差距需要重构多少内容？
            ├── 半章以内 → P1
            └── 超过半章 → 问：不改的话后续章节会崩吗？
                  ├── 会 → P0
                  └── 不会 → P2 / 保持
```

## 自动修改流程

### 修改原则

1. **保留核心** - 不改变种子核心欲望、人物核心设定
2. **逐项优化** - 按照优先级逐一修改
3. **可追溯** - 每处修改标注原因
4. **可逆性** - 修改后效果不佳可还原

### 修改执行

```
【自动修改执行】

对于每个P0/P1级别的差距：

```

## 使用方式

### 触发方式

用户输入以下命令即可触发：

- "对比" + 自己的网文 + "和大神的[大神名]"
- "分析差距" + 自己的网文
- "帮我改进" + 自己的网文 + "参考[大神名]"
- "学习[大神]" + 自己的网文

### 输入要求

```
【必填】
- 自己的网文（章节/段落）

【可选】
- 指定大神：提到具体大神名字，或说明想学习的大神风格
- 对比维度：说明重点关注哪些维度
- 修改强度：轻微调整/中等修改/大幅重构
```

### 输出承诺

- 提供详细的对比分析报告
- 针对每个核心差距提供具体的修改方案
- 直接输出优化后的完整版本
- 提供后续学习建议

## 注意事项

1. **尊重原创** - 修改是在学习大神技巧，不是一味模仿
2. **保持特色** - 自己的优点要保留，只改不足
3. **分步验证** - 修改后先自检，再决定是否采纳
4. **风格适配** - 选择适合自己的大神，不要盲目跟风
5. **持续迭代** - 对比改进是循环过程，需要多次迭代
