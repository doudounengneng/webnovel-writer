## 自动Handoff协议

每个阶段完成时，自动移交主导权：

| 当前 | → 下一 | 主导权移交 | 传递摘要 |
|------|--------|-----------|---------|
| 阶段0 | 阶段1 | 十层宇宙 → 墨白 | M层[L6/L7] |
| 阶段1 | 阶段2 | 墨白 → 十层+创世 | 复仇者，系统流，L6打破 |
| 阶段2 | 阶段3 | 十层+创世 → 创世神 | ⚔️引爆点第X章 |
| 阶段3 | 阶段4 | 创世神 → 墨白 | 三线已锁定 |
| 阶段4 | 阶段5 | 墨白 → 网文大师 | 人设完整 |
| 阶段5 | 阶段6 | 网文大师 → 墨白 | 草稿完成 |
| 阶段6 | 阶段7 | 墨白 → 网文大师 | 校对完成 |

**Handoff消息格式**：
```
【Handoff】阶段X → 阶段Y

主导权从 [Skill A] 移交至 [Skill B]

📋 传递摘要：
[50字内核心信息]

📌 下一阶段任务：
[任务描述]

✅ 质量门：已通过

[Skill B] 请接管...
```

### 协作流程

```
你（在窗口1 CEO中）：
  创世 穿越洪荒世界主角拜入阐教

CEO自动：
  → 窗口6：调研 同类型爆款
  → 窗口2：十层 基于调研分析M层
  → 窗口3：世界观 基于M层搭建世界
  → 窗口4：人设 基于世界观灌注人设

  → 窗口5：网文大师 定稿

```
