# 创世工作流引擎

> 一键启动 · 自动协作 · 阶段流转

---

## 一键启动

```
输入：创世 [灵感/主题]
例如：创世 穿越洪荒世界，主角拜入阐教受偏见
```

## 自动协作协议

### Skill主导权移交（Handoff）

| 当前阶段 | 主导Skill | 协作Skill | 移交时机 |
|---------|----------|-----------|---------|
| 阶段0 | 十层宇宙 | 墨白 | 五维定义完成后 |
| 阶段1 | 墨白 | 十层宇宙 | 核心心跳确认后 |
| 阶段2 | 十层宇宙 | 创世神 | 引爆点锚定后 |
| 阶段3 | 创世神 | 墨白 | 世界观完成，交给人设 |
| 阶段4 | 墨白 | 网文大师 | 人设完成，交给正文 |
| 阶段5 | 网文大师 | 墨白 | 草稿完成，交给校对 |
| 阶段6 | 墨白 | 网文大师 | 校对完成，交给定稿 |

### Handoff消息格式

```
【Handoff】阶段X → 阶段Y

主导权从 [Skill A] 移交至 [Skill B]

📋 传递摘要（50字内）：
[核心信息1] + [核心信息2] + [核心信息3]

📌 下一阶段任务：
[任务描述]

✅ 质量门检查：
- [ ] 检查项1
- [ ] 检查项2

[Skill B] 请接管并继续...
```

## 工作流命令

| 命令 | 功能 |
|------|------|
| `创世 [灵感]` | 一键启动完整工作流 |
| `继续` | 从暂停点继续 |
| `跳过` | 跳过当前阶段（需确认） |
| `暂停` | 在当前暂停点暂停 |
| `状态` | 查看当前工作流状态 |
| `锚点` | 查看当前锚点摘要 |
| `重置` | 重置工作流（从头开始） |

## 自动质量门检查

每个阶段切换时自动检查：

```
【质量门检查】阶段3 → 阶段4

主导Skill：创世神 → 墨白

✅ 世界观逻辑闭环
✅ 修炼体系完整
✅ 三线并行不冲突
✅ 长线伏笔≥3
✅ 爽点密度规划明确

【检查结果】✅ 全部通过，可以移交
```
