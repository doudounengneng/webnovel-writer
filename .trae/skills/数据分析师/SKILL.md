## 追读率分析

### 追读率计算模型

```
章节追读率 = 本章读者数 / 上一章读者数 × 100%

健康标准：
  前10章：追读率>60%（留存期）
  10-50章：追读率>40%（稳定期）
  50章+：追读率>30%（成熟期）

预警阈值：
  单章追读率下降>10% → 检查该章质量
  连续3章追读率下降 → 剧情出问题
  追读率<20% → 需要大调整
```

### 流失章节定位

```
【流失分析报告】

章节X追读率：从45%降至32%（下降13%）

可能原因（按排查优先级排列）：
  1. 🔴 节奏拖沓（废话段落过多）         ← 先查这个，最常见
  2. 🔴 爽点不足（该章无核心爽点）         ← 第二常见
  3. 🟡 剧情平淡（无推进/无冲突）         ← 前两个排除后再查
  4. 🟡 人物OOC（行为不符合人设）          ← 最后查，相对少见

排查步骤：
  Step 1：用"三句话测试"——这章去掉后，读者是否完全能接上？能→节奏拖沓
  Step 2：找出本章最大的爽点——如果没有→爽点不足
  Step 3：列出本章对主线/暗线/人物弧的推进——如果一条都没有→剧情平淡
  Step 4：逐行为检查是否符合人物灵魂卡→如有明显违背→OOC

建议：
  1. 检查该章爽点密度
  2. 对比前后章节质量
  3. 读者评论分析
```

## 付费转化分析

### 转化率模型

```
付费转化率 = 付费读者数 / 免费读者数 × 100%

健康标准：
  免费→VIP首章：转化率>15%
  VIP持续订阅率：>60%
  单章付费率：>8%

优化方向：
  转化率低 → 检查免费章节结尾钩子
  订阅率低 → 检查VIP章节质量
  单章付费低 → 检查付费点设计
```

### 画像应用

```
基于读者画像的创作调整：

年龄偏年轻（18-25岁占60%+）：
  → 节奏更快，爽点更密集
  → 感情线更甜

性别偏男性（男占70%+）：
  → 升级/战斗/权谋为主
  → 感情线为辅

阅读时段偏夜间（22-00点占40%+）：
  → 章末钩子要强
  → 适合睡前阅读的节奏
```

## A/B测试

### 测试维度

| 测试项 | 测试方法 | 评估指标 | 测试周期 |
|--------|---------|---------|---------|
| **书名** | 不同平台用不同书名 | 点击率 | 1-2周 |
| **简介** | 同一平台AB版简介 | 收藏率 | 1周 |
| **开篇** | 不同开篇版本 | 追读率 | 3天 |
| **封面** | 不同风格封面 | 点击率 | 1周 |
| **更新时间** | 不同时段发布 | 追读率 | 1周 |

### A/B测试流程

```
【A/B测试流程】

Step 1：确定测试目标
  例：提升首章追读率

Step 2：设计两个版本
  A版：当前版本（对照组）
  B版：优化版本（实验组）

Step 3：设置测试
  同一平台
  同一时间段
  样本量：各1000个曝光

Step 4：收集数据
  测试周期：3-7天
  数据指标：点击率/追读率/收藏率

Step 5：分析结果
  B版胜出 → 全量切换
  A版胜出 → 保持现状
  无显著差异 → 继续测试

Step 6：输出报告
  [A/B测试报告]
```

## 收入预测模型

### 单本书收入预测

```
【收入预测模型】

输入参数：
  日均新增收藏：[X]
  追读率：[X]%
  付费转化率：[X]%
  ARPU（每用户平均收入）：[X]元
  预计总字数：[X]万字
  预计更新天数：[X]天

预测公式：
  日均收入 = 日均新增收藏 × 追读率 × 付费转化率 × ARPU
  总收入 = 日均收入 × 预计更新天数

示例：
  日均收藏：200
  追读率：35%
  付费转化率：12%
  ARPU：15元/月
  预计更新：180天

  日均收入 = 200 × 35% × 12% × (15/30) = 4.2元
  总收入 = 4.2 × 180 = 756元（保守估计）
```

### 多开收入预测

```
【多开收入预测】

书A（主书）：日均收入[X]元 × 180天 = [X]元
书B（副书1）：日均收入[X]元 × 180天 = [X]元
书C（副书2）：日均收入[X]元 × 180天 = [X]元

总收入 = 书A + 书B + 书C

多开收入倍增条件：
  1. 主书稳定后（追读率>30%）再开副书
  2. 副书选择不同类型（分散风险）
  3. 副书采用"低维护"模式（日更1章即可）
```
