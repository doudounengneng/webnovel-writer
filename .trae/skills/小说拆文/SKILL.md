# 小说拆文

深入拆解小说结构，提取故事核心要素，为创作提供参考或学习模板。

## 核心拆解维度

### 1. 基本信息

| 维度 | 说明 |
|------|------|
| 故事类型 | 玄幻/都市/穿越/科幻/悬疑/言情等 |
| 角色标签 | 主角/配角/反派等定位 |
| 情节标签 | 升级流/复仇流/甜宠/虐恋等 |
| 情绪标签 | 热血/治愈/压抑/爆笑等 |
| 中心思想 | 故事核心表达的主题 |
| 世界观 | 故事背景的宏观设定 |
| 故事主线 | 核心叙事脉络 |

### 2. 故事背景

- **时代背景**：故事发生的时间节点
- **社会环境**：政治、经济、文化等社会形态
- **主要势力**：涉及的核心势力或组织
- **核心冲突**：故事的主要矛盾来源

### 3. 情感线

- **起始**：情感线如何开端
- **承接**：情感如何发展推进
- **转折**：情感的关键转折点
- **高潮**：情感爆发的核心节点
- **结局**：情感线的最终走向

### 7. 金手指/系统设定

- **金手指类型**：系统/老爷爷/特殊体质/知识碾压等
- **能力来源**：如何获得/触发条件
- **能力边界**：能力的限制或代价
- **升级逻辑**：能力如何成长进阶

### 8. 修炼体系/能力等级

- **等级划分**：各阶段等级名称（如炼气/筑基/金丹等）
- **突破条件**：升级所需条件
- **战力表现**：不同等级的战力差距
- **体系特点**：该体系的独特之处

## 使用方式

用户输入"拆文"或"分析小说"，然后提供小说文本或描述，即可获得完整的拆解报告。

## 注意事项

1. 拆解时保持客观，提取原文核心信息而非过度解读
2. 人物设定尽量还原原文描述
3. 情感线和情绪拉扯要标注具体的章节/段落位置
4. 世界设定注意前后一致性
5. 金手指和修炼体系是网文核心，需重点拆解
6. 伏笔需标注埋设和回收的具体位置
7. 爽点分布注意统计密度和类型比例
