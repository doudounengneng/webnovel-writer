---

## 阶段1：灵魂激活

**任务**：追问核心心跳

**必须确认**：
- 主角真正想要的（核心欲望）
- 最害怕失去的
- 金手指类型
- 世界底色
- 打破层 + 代价

---

## 阶段4：灵魂灌注

**任务**：人设立体化

**主角必须包含**：
- 性格标签（3-5个关键词）
- 核心欲望（一句话，不可优化）
- 成长弧光
- 高光锚点（第X章）
- 底线锚点（绝对不会做的事）

**配角必须包含**：
- 独立动机（这个角色自己想要什么）
- 在故事中的作用
- 高光锚点
- 底线锚点

**反派必须包含**：
- 独立权谋暗线
- 反派自己的目标
- 高光锚点

**感情线**：
- 升温节点规划
- 克制但到位

### 阶段4扩展：扩写能力注入

人设完成后，Handoff时向网文大师注入防短文指令：

```
📋 本章写作风格指令：
1. 章节结构锚点：必须覆盖5要素（环境/动作/对话/内心/爽点）
2. 三感法：每场景至少2种感官描写
3. 对话扩写：每段对话前加动作描写
4. 事件三段式：重要事件分铺垫/过程/结果三段
5. 字数目标：4000-6000字
```

2. 对话是否像真人？
3. 人物行为是否有情感支撑？
4. 有无"完美台词"？
