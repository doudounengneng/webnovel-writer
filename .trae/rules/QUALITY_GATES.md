# 质量门清单

> 参考vibewriting的契约验证机制，为每个阶段定义可执行的质量门检查表

---

## 一、质量门概述

### 质量门层级

| 层级 | 适用对象 | 检查时机 | 失败后果 |
|------|---------|---------|---------|
| **十层宇宙质量门** | 阶段0-2 | 每个阶段结束时 | 必须修正才能进入下一阶段 |
| **创世神质量门** | 阶段3 | 阶段结束时 | 必须修正才能进入下一阶段 |
| **墨白质量门** | 阶段1/4/6 | 阶段结束时 | 必须修正才能进入下一阶段 |
| **网文大师质量门** | 阶段5/7 | 阶段结束时 | 必须修正才能进入下一阶段 |
| **全局质量门** | 所有阶段 | 任意时刻 | 警告，不阻止阶段切换 |

---

## 二、十层宇宙质量门

### 阶段0：灵感激活

```markdown
## 十层宇宙质量门 - 阶段0

### 核心检查
- [ ] M层数量 ≤ 3
- [ ] M层都有对应的故事场景/角色欲望支撑
- [ ] 每个M层都有五维定义
- [ ] 每个五维都有命运标签

### 五维质量检查
- [ ] 道的定义抓住了层的"根"
- [ ] 法的定义是道的延伸
- [ ] 术有具体的操作描写
- [ ] 器有明确的承载物
- [ ] 势有清晰的矛盾方向

### 命运标签质量检查
- [ ] 🧠 内功：确实不需要在正文中出现
- [ ] 👁️ 暗示：有具体的意象/细节设计
- [ ] ⚔️ 情节引爆：具体到章节号

### U层质量检查
- [ ] U层数量 ≥ 5（保证惊喜空间）
- [ ] 没有强行填充U层
- [ ] U层有"未知感"而不是"空白感"

### M层与种子匹配检查
- [ ] 种子确实触及了每个M层
- [ ] 没有多余的M层
- [ ] 打破层的选择有故事逻辑支撑

### 向下传递检查
- [ ] 向下传递摘要 ≤ 50字
- [ ] 摘要包含：M层清单 + 五维核心 + 种子描述
```

### 阶段2：骨架激活

```markdown
## 十层宇宙质量门 - 阶段2

### 五维细化检查
- [ ] 每个M层都有完整的五维
- [ ] 五维之间逻辑自洽
- [ ] 没有自相矛盾的描述

### 命运标签检查
- [ ] 每个M层至少1个⚔️情节引爆点
- [ ] 情节引爆点具体到章节号
- [ ] 情节引爆点与主线走向一致

### 世界观框架检查
- [ ] 三界/种族/势力格局有一句话描述
- [ ] 修炼体系有一句话描述
- [ ] 天道规则有一句话描述

### 向下传递检查
- [ ] 向下传递摘要 ≤ 50字
- [ ] 摘要包含：⚔️引爆点 + 世界框架
```

## 三、创世神质量门

### 阶段3：骨架搭建

```markdown
## 创世神质量门 - 阶段3

### 世界观逻辑检查
- [ ] 世界观逻辑自洽，无明显漏洞
- [ ] 修炼体系完整：境界名称/突破条件/功法分类
- [ ] 势力格局清晰：主角势力/对立势力/第三方
- [ ] 天道规则有约束力（不是可以随意打破的）

### 伏笔检查
- [ ] 长线伏笔（卷级/书级）≥ 3个
- [ ] 中线伏笔（章级/情节级）≥ 5个
- [ ] 每个伏笔都有明确的埋设章节
- [ ] 每个伏笔都有明确的回收章节
- [ ] 伏笔之间不冲突

### 三线并行检查
- [ ] 主线明线：有明确的成长路径
- [ ] 反派暗线：反派有自己的独立计划
- [ ] 身份秘线：主角身份秘密有铺垫
- [ ] 三线交织节点明确
- [ ] 三线不会相互冲突

### 爽点密度检查
- [ ] 明确的爽点密度规划
- [ ] 章小爽：每章都有
- [ ] 三章中爽：有具体节点
- [ ] 十章大爽：有具体节点
- [ ] 爽点类型分布合理（打脸/升级/情感/智谋）

### 节奏检查
- [ ] 无废话章节
- [ ] 每章都有推进
- [ ] 章末钩子类型多样
- [ ] 节奏紧凑无拖沓

### 输出格式检查
- [ ] 世界观设定.md结构完整
- [ ] 三线设计.md三线清晰
- [ ] 伏笔清单.md表格规范
```

## 四、墨白质量门

```markdown
- [ ] 主角最害怕失去的有明确的句子

- [ ] 金手指类型明确
- [ ] 金手指机制清晰
- [ ] 金手指不会破坏故事平衡

- [ ] 整体氛围有描述词（黑暗/光明/灰色等）
- [ ] 叙事基调和氛围一致
- [ ] 不会中途改变基调

### 打破层检查
- [ ] 打破层有明确的选择理由
- [ ] 打破代价有具体描述

- [ ] 不是脸谱化人设
```

### 阶段4：灵魂灌注

```markdown
## 墨白质量门 - 阶段4

### 主角人设检查
- [ ] 性格标签 ≥ 3个关键词
- [ ] 核心欲望有明确描述

- [ ] 底线锚点明确

- [ ] 每个重要配角都有独立动机
- [ ] 配角不是工具人
- [ ] 配角有高光锚点
- [ ] 配角底线明确

### 反派人设检查
- [ ] 反派有独立权谋暗线
- [ ] 反派不是纯粹的坏人
- [ ] 反派有自己的目标
- [ ] 反派高光锚点明确

### 感情线检查
- [ ] 感情线有升温节点
- [ ] 感情线不喧宾夺主
- [ ] 情感拉扯细腻克制
- [ ] 感情线有考验节点

### 活人感自审
- [ ] 每个角色都有独特的声音
- [ ] 对话不像教科书
- [ ] 人物行为有情感支撑
- [ ] 无明显工具人风险
```

## 七、质量门执行流程

### 自动执行时机

1. **阶段切换时**：执行该阶段的质量门
2. **暂停点确认前**：执行全局质量门
3. **断点续跑时**：检查项目状态与质量门的一致性

### 质量门报告格式

```markdown
【质量门检查报告】

阶段：阶段5（正文输出）
检查时间：2026-05-04 20:30

## 检查结果汇总
- 核心检查：✅ 通过
- 五维质量：✅ 通过
- 命运标签：⚠️ 警告（见详情）
- U层质量：✅ 通过

## 详细结果

### ✅ 通过项
- 节奏检查：✅ 每章都有推进
- 爽点兑现：✅ 打脸爽点即时
- 章末钩子：✅ 悬念型钩子

### ⚠️ 警告项
- M层描写：⚠️ L7层的社会描写稍弱
  - 建议：增加势力冲突场景

### ❌ 未通过项
- 伏笔处理：❌ 第8章埋设的伏笔未在第15章回收
  - 建议：在第16章安排回收

## 总体评估
状态：⚠️ 有条件通过
建议：修复伏笔问题后再进入阶段6

是否继续？ [修复后继续/有条件继续/暂停]
```

## 八、质量门与契约的关系

| 质量门 | 对应契约 | 检查内容 |
|-------|---------|---------|
| 十层宇宙质量门 | 种子冲击报告/骨架激活报告 | M层定义、五维、命运标签 |
| 创世神质量门 | 世界观设定/三线设计/伏笔清单 | 世界观逻辑、伏笔完整性 |
| 墨白质量门 | 灵魂激活卡/人物灵魂卡/校对报告 | 人设立体化、活人感 |
| 网文大师质量门 | 章节正文/定稿 | 节奏爽点、错别字 |
| 全局质量门 | 所有文件 | 锚点一致性、轨迹追踪 |
