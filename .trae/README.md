# Webnovel-Writer Trae Skill Package

> Import this package into Trae IDE to unlock 11 specialized AI agents for web novel creation.

## Installation

1. Clone the repository
2. Open the project folder in Trae IDE
3. The `.trae/` directory is automatically loaded
4. Start with the `创世` command

## Quick Start

```
创世 [你的灵感]
```

Example:
```
创世 穿越洪荒世界，主角拜入阐教受偏见
```

## Available Commands

| Command | Function |
|---------|----------|
| `创世 [灵感]` | Start full workflow |
| `十层 [种子]` | Activate Planning Dept (M-Layer analysis) |
| `世界观 [设定]` | Activate Technology Dept (world-building) |
| `人设 [大纲]` | Activate Editorial Dept (character design) |
| `写 [章节]` | Activate Production Dept (chapter writing) |
| `拆解 [书名]` | Activate Market Research (competitive analysis) |
| `对比 [章节]` | Activate QA (quality comparison) |
| `运营 [作品]` | Activate Operations (platform strategy) |
| `分析 [作品]` | Activate Data Analytics (data analysis) |
| `营销 [作品]` | Activate Content Marketing (promotion) |
| `IP评估 [作品]` | Activate IP Development (IP valuation) |
| `继续` | Resume from pause point |
| `状态` | View workflow status |
| `锚点` | View anchor summary |

## 8-Stage Workflow

```
Stage 0: 灵感激活 (十层宇宙)
Stage 1: 灵魂激活 (墨白)
Stage 2: 骨架激活 (十层+创世)
Stage 3: 骨架搭建 (创世神)
Stage 4: 灵魂灌注 (墨白) ← pauses for confirmation
Stage 5: 正文输出 (网文大师)
Stage 6: 灵魂校对 (墨白)
Stage 7: 定稿交付 (网文大师)
```
